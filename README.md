# Console-Number-Game
记录自学C语言和C++语言时曾做过的几个由数字驱动的控制台小游戏

> [!NOTE]
> 如果你问为什么这个项目html代码更多？
> 
> 那是因为ai对html的重构的每一代版本都有各自比较特别的一面，所以我把他们全部保留了下来，所以html代码会比C和C++代码多得多（并不仅仅是因为html代码本身比C和C++长）。

其中包含了三个游戏：

- numberfinding —— 数轴探宝
- passwordcracking —— 密码破解
- wateroverflow —— 倒水挑战

## 初步介绍

* 》 numberfinding —— 数轴探宝

  原名：numberboom，数字炸弹。

  正如原名，该游戏是参考“数字炸弹”做的小游戏。最初一代版本1:1还原了数字炸弹原游戏（不过是在控制台中），后来的版本增加了一些全新的功能，4.0版本开始改名“numberfinding（数轴探宝）”。

  其中，一代、二代、三代、四代均为C语言代码，第五代开始改用C++。

* 》 passwordcracking —— 密码破解

  参考玩具“超级密码机”制作的游戏。

  其中，一代、二代为C语言代码，第三代开始改用C++，除此之外，还有借助deepseek制作的网页版。

* 》 wateroverflow —— 倒水挑战

  参考“由多个人依次上前按按钮，一个人一次可以按多次按钮，导致与按钮连接的炸弹爆炸的人失败”这样的小游戏制作的游戏，不过把按按钮改成了倒水。

  游戏包含两个C++版本和借助deepseek制作的网页版（包括人工二次微调版）。

## 使用教程

想要在自己的电脑上使用这些游戏很简单，只需要在Visual Studio上自己新建一个新项目，然后将代码复制粘贴进去即可。

* 如果你问其他工具是否可以？

  至少我没用过，理论上，只要能编译运行C语言或C++代码应该都可以，不过Visual Studio应该是比较简单的一个方案，因为他会帮你准备好必要的依赖，而在像VSCode的地方，需要自行处理依赖。
